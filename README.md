# flappyBirdForEdu
Flappy Bird 클론 코딩

전 세계적으로 큰 인기를 끌었던 모바일 게임 **플래피 버드(Flappy Bird)** 의 게임을 클론 코딩 합니다. [플레이 영상](https://youtu.be/HzvpOtyJYGs?si=lxTMQaofLSGTtHeC) 참고해 웹에서 할 수 있는 게임을 제작합니다. GitHub Pages 활용해 퍼블리싱할 예정입니다.

[스크린샷](screenshot/) 폴더 참고해 제작합니다. 모든 리소스는 이미지 없이 도트로 제작합니다.

## 파일 구조

초등학생도 쉽게 열어서 수정할 수 있도록 HTML/CSS/JS를 `index.html` 한 파일에 모두 합쳐서 작성합니다. `<style>` 태그 안에 스타일, `<script>` 태그 안에 게임 로직이 들어 있습니다. 사운드 파일만 `sounds/` 폴더에 따로 둡니다.
